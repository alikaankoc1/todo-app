# todo-app

Basit, modern ve şık bir React Todo uygulaması.

## Özellikler

- Görev ekleme ve silme
- Görevleri düzenleme (ikon ile edit/check)
- Modern responsive arayüz
- React ile component tabanlı yapı

## Kurulum

1. Bu repoyu klonlayın veya indirin.
2. Terminalde proje klasörüne girin:
   ```bash
   cd todo-app/todo-app
   ```
3. Gerekli paketleri yükleyin:
   ```bash
   npm install
   ```
4. Uygulamayı başlatın:
   ```bash
   npm run dev
   ```

## Kullanılan Teknolojiler

- React
- Vite
- react-icons
- CSS (modern, responsive tasarım)
